What this project is about

In this learning project, I test various technologies, which help me build a rather complex webapp prototype. The webapp is based on AngularJS (for now).

The basic ideas of this webapp are:

  • Reuse as much as possible instead of reinventing.
  • Be lean by reusing what is available and apply basic patterns and practices.
  • Do not just use a technology to be cool - use it the right way.
  • Learn from mistakes and do it better next time.

Though, the content of the webapp itself will not crave your attention, the setup of it will. It uses

  • a lean but stable ES6+ modularized project setup,
  • Webpack as a bundler,
  • Babel as a JavaScript "compiler",
  • Karma and Jasmine for a unit test environment,
  • AngularJS, UI-Router and AngularJS Material as the base of the ui,
